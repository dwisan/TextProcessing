# grep Command In Linux
- [x] Syntax
```
grep [option] pattern [files]
```
> Case insensitive search
```
grep [option] pattern [files]
```
> Displaying the count of number of matches
```
grep [option] pattern [files]
```
> Display the file names that matches the pattern
```
grep [option] pattern [files]
```
> Checking for the whole words in a file
```
grep [option] pattern [files]
```
